# B.B. AIRCONDITIONS WEBSITE — REPLIT PROMPT

## PROJECT BRIEF
Single-page HTML website for B.B. Airconditions, a Daikin-authorized AC contractor in Kolkata. Target: both residential and commercial clients. 

---

## DESIGN TOKENS

**Color Palette:**
- Dark Navy: #1a2a3a (primary bg)
- Daikin Blue: #0099cc (accents, hover states, CTAs)
- Light Navy: #2d3e4f (footer, alternate sections)
- White: #ffffff (text, cards)
- Green (WhatsApp): #25d366
- Light BG (Brands section): #f5f5f5

**Typography:**
- Headings: Inter Bold or Barlow Condensed Bold (Google Fonts)
- Body: Inter Regular or Open Sans (Google Fonts)
- No serif fonts

**Spacing & Layout:**
- Consistent padding: 40px horizontal, 60px vertical per section
- Mobile: 20px horizontal, 40px vertical
- Border radius: 8px for cards, 12px for buttons
- No hard-cut transitions between sections — soft visual separation only

**General Rules:**
- No AI-generated look, no generic templates
- All text is real, specific to the business
- Mobile responsive from the start
- Fast load time — no unnecessary animations or heavy assets

3. **WhatsApp link format:** `https://wa.me/918777793800?text=I%20need%20AC%20services`
4. **Phone links format:** `tel:+918777793800`
5. **Email link format:** `mailto:ankan@novasites.co`
6. **No framework code, no React, no Vue** — vanilla HTML/CSS/JS only
7. **Tailwind CSS via CDN only** — no npm installs
8. **All text must be actual copy, no placeholders**
9. **Icons:** Use Lucide React via CDN (lucide.dev) or simple SVG inline
10. **Performance:** Page should load and feel snappy — no heavy scripts
11. **Daikin logo:** Use official logo from Daikin's brand site (freely available)
12. **Godrej logo:** Use official logo from Godrej's brand site
13. **No AI-generated look** — clean, professional, purpose-built
